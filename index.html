What

Monitor a child process and restart it if required.

Features

Current

  • Run in the foreground or as a daemon.
  • Writes child process stdout and stderr to syslog when running as a daemon.
  • Delay between restarts, with backoff.
  • Set environment variables before running the child.
  • Clear the child process environment (before setting the specified variables.)
  • Run the child process with a different uid and gid.
  • Passes appropriate signals to the child process (SIGHUP, SIGINT, SIGTERM.)
  • Can be told to stop monitoring the child process, and to re-start monitoring it later.

Upcoming

  • Send email when child exits or is restarted.
  • Autoconf?
